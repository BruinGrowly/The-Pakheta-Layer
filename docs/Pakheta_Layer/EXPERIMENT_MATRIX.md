## 4. Experiment Families

### 4.1 Anchor Coherence Experiments

Question:

```text
When is one anchor enough?
```

Variables:

- number of nodes
- number of anchors
- anchor-to-field alignment
- facet coverage
- false partition penalty
- complementary differentiation

Expected result:

```text
one-field-aware anchors > false partition anchors
```

### 4.2 Context Actualization Experiments

Question:

```text
Can many actualized meanings come from one stable field?
```

Variables:

- context vector
- attention weight
- ritual/action weight
- memory intensity
- anchor stability

Expected result:

```text
same field + different context -> different state, same identity
```

### 4.3 Relational Distance Experiments

Question:

```text
Can relational nearness be independent of physical nearness?
```

Variables:

- physical graph distance
- semantic vector distance
- field-mediated distance
- shared anchor count
- memory/history overlap

Expected result:

```text
field-mediated distance predicts coherence better than physical distance
```

### 4.4 Phi Coherence Experiments

Question:

```text
Does phi preserve differentiated unity better than arbitrary proportion?
```

Variables:

- phi weights
- equal weights
- random weights
- over-fused weights
- over-split weights

Expected result:

```text
phi-normalized weighting improves coherence when a field must remain one while expressing many facets
```
